# App Design V5 - Projektna dokumentacija

Ovaj folder sadrži sve stavke vezane za UX/UI dizajn aplikacije (V5 dizajn sustav - Future Light).

## Struktura foldera:
- **01-research/**: Istraživanja tržišta, konkurencije i UX trendova.
- **02-screens/**: Opisi i specifikacije pojedinačnih ekrana aplikacije.
- **03-flows/**: Dijagrami i opisi korisničkih putova (user flows).
- **04-components/**: Specifikacije UI komponenti (gumbi, kartice, input polja).
- **05-assets/**: Ikone, slike i ostali grafički elementi.
- **06-functionalities/**: Popis potvrđenih funkcionalnosti po ekranima.

## Trenutni fokus:
Priprema okruženja za integraciju novih ekrana i funkcionalnosti na temelju postojećeg dizajna.
